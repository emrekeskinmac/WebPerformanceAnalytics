# Start WebAPI

```
yarn api
```
[API Swagger](https://127.0.0.1:3000/swagger/)

Seed endpointinde "size" değerine istediğiniz bir değer girdikten sonra seed data oluşturabilirsiniz.

# Start Dashboard

```
yarn web
```

[Dashboard](http://localhost:8080/)
